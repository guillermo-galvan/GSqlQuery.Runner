# GSqlQuery.Runner [![es](https://img.shields.io/badge/lang-es-red.svg)](./README.es.md)[![NuGet](https://img.shields.io/nuget/v/GSqlQuery.Runner.svg)](https://www.nuget.org/packages/GSqlQuery.Runner)
